时尚俚语

completely fair(ly), juestly, in an honest wa and without any doubt.

中文意思是：公平，公正，光明正大

例句：

1.We won that game fair and square.

那场比赛我们赢得光明正大。

2.I’d rather los fair and squase than win with a dirty campaign.

我宁愿堂堂正正地输，也不要耍阴谋来赢得选举。

2.What would you like for your main course/side dishes ?

主菜/配菜你想要点什么？

你是喜欢套餐还是自己点菜？

口语小词大用

start的高频口语用法

1.开始

Let’s start in 3.

我们三分钟后开始。

I want a new start.

我想要一个新的开始。

2.重新开始(start over)

You can start over.

你可以重新开始啊。

Move to a new city and start everything over.

搬到一个新的城市然后一切从新开始。

3首先（to start with）

Ok, to start with, you’re not my type.

好吧，首先，你不是我喜欢的类型。

To start with, American English is more popular.

首先，美语更受欢迎一点。

 

每日一句

If you feel you have no fault, that make another one.

如果你觉得自己没有错误，那你就犯了一个错误。

 

美剧口语精讲精练

what’s with the hat?

干嘛戴那个帽子？

关键短语what’s with …表示 “干嘛…”

例句：

1.What’s with the car ?

干嘛开车来？

2.What’s with the red T-shirt?

干嘛穿着红T恤？
